# Skill Creator

Turn the requested workflow into a small, testable skill.

- Define a precise trigger and explicit non-triggers.
- Keep `SKILL.md` focused; place substantial examples or scripts in referenced files.
- Preserve approval, credential, ownership, and side-effect boundaries from the underlying task.
- Reuse repository conventions and validate every referenced path.
- Exercise representative positive and negative triggers before declaring the skill ready.
